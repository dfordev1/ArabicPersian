# Lovable Prompt

Build an offline multilingual Qur'an app using local JSON files.

Languages:
Turkish, Azerbaijani, Kazakh, Kyrgyz, Uzbek.

Features:
- language switcher
- surah reader
- Arabic-script rendering
- original translation
- footnotes
- dictionary lookup
- full search
- bookmarks
- dark mode
- offline first

Do not use online APIs.
Do not create mock data.
Use real JSON files from /assets/data/languages/.
