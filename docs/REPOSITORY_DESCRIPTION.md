# Repository Description

ArabicPersian is a multilingual Arabic-script Qur'an rendering repository built
to grow across multiple languages over time.

## Suggested GitHub description

```text
Multilingual Arabic-script Qur'an renderings with release bundles, QA tooling, and an expandable app scaffold for Turkish, Azerbaijani, Kazakh, and future languages.
```

## Suggested scope summary

- Preserve source release ZIPs and pipeline intermediates
- Add new languages without reshaping the repo
- Keep app data derived from the release bundles
- Avoid official or scholarly-verified claims unless separately reviewed
